# learning-moments

Learning Moments
An application built with React. Allows users to create posts, share posts with other users, and favorite programming concepts posted by other users. Allows for post editing and profile editing. 

Features:
Authentication
Create, edit, and delete posts
Posts organized by topic, can search and filter by title and topic
Like posts to add to favorites, can remove posts from favorites
User profiles with post counts, allows for user to edit their own profile

Technologies utilized:
React
JSON Server (RESTful API)
CSS
JavaScript

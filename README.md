<h1>Learning Moments</h1>
<p><img src="https://raw.githubusercontent.com/judeottilie-tech/learning-moments/refs/heads/judeottilie-tech-patch-1/Animation.gif"/></p>
An application built with React. Allows users to create posts, share posts with other users, and has the ability to favorite programming concepts posted by other users. Lets the user edit their own posts and profile.

<div><p><h2>Features:</h2></p>
<p>Authentication</p>
<p>Create, edit, and delete posts</p>
<p>Posts organized by topic, can search and filter by title and topic</p>
<p>Like posts to add to favorites, can remove posts from favorites</p>
<p>User profiles with post counts, allows for user to edit their own profile</p></div>
<div><p><h2>Technologies utilized:</h2></p>
<p>React</p>
<p>JSON Server (RESTful API)</p>
<p>CSS</p>
<p>JavaScript</p></div>

<h3>for running locally:</h3>

<div><p>git clone (URL github gives you for this project here)</p>
<p>npm install</p>
<p>npm run dev</p>
<p>cd to the api, then:</p>
<p> json-server -p 8088 -w database.json</p></div>
